# Технические требования — игра "Лабиринт"

## Версии игры

- **`start3.py`** — спираль Архимеда, работа на двух мониторах/проекторах (игра на втором экране 1920×1080)
- **`start4.py`** — спираль Архимеда, работа в одном окне на основном мониторе (разрешение определяется автоматически)

---

## Операционная система

- **Минимально:** Windows 7 SP1 (32/64-bit)
- **Рекомендуется:** Windows 10 или 11 (64-bit)

Причины:
- Используется `pywin32` (`win32gui`, `win32con`) — кроссплатформенность не предусмотрена
- Современные версии Python (3.9+) не поддерживают Windows XP/Vista
- Драйверы сканера ZKFinger SDK официально поддерживают Win7/8/10/11, для ARM-версий Windows драйверов нет (нужен x86/x64)

---

## Аппаратное обеспечение

### Общее
- ПК с x86/x64 процессором (требования к производительности минимальны — 2D-графика, лёгкая физика pymunk)
- Цифровая клавиатура (NumPad): клавиши `KP4`/`KP6` — управление вращением лабиринта и аварийный сброс (тройное одновременное нажатие в течение 3 сек)
- USB-сканер отпечатков пальцев **ZKTeco** (совместимый с ZKFinger SDK, `libzkfp.lib`)

### Для `start3.py` (два экрана)
- Два монитора/проектора:
  - Левый — для `Demo.exe` (управление сканером)
  - Правый — разрешение **1920×1080**, на нём запускается игра в полноэкранном режиме без рамки

### Для `start4.py` (один экран)
- Один монитор — игра запускается без рамки на весь экран, разрешение определяется автоматически

---

## Программное обеспечение

1. **Windows** (см. выше)
2. **Драйверы и SDK ZKFinger** — для работы биометрического сканера
3. **Demo.exe** (приложение на Delphi, `Demo.dpr`/`Unit1.pas`):
   - принимает данные со сканера, сохраняет снимок отпечатка в `Fingerprint.jpg`/`Fingerprint.bmp`
   - должен быть запущен до старта игры (фоновый поток игры сам найдёт окно "Demo" и нажмёт кнопку "Save Image" через `PostMessage`, даже если окно скрыто)
4. **Python 3.8+** с пакетами:
   - `pygame` — графика и игровой цикл
   - `pymunk` — 2D физический движок
   - `opencv-python` (`cv2`) — анализ изображения отпечатка
   - `numpy` — обработка массивов изображения
   - `pywin32` (`win32gui`, `win32con`) — автоматизация окон Demo.exe

---

## Требования к файлам

- Игровой скрипт (`start3.py` или `start4.py`) и `Fingerprint.jpg` должны находиться в одной папке
- `Fingerprint.jpg` обновляется приложением `Demo.exe` после сканирования отпечатка; от количества пересечений линий на отпечатке зависит сложность (число витков) спирали лабиринта
